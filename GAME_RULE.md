# 게임 기획서: Fruit Catcher (과일 받기 게임)

## 1. 게임 개요
하늘에서 떨어지는 과일을 바구니로 받아 점수를 획득하는 게임입니다. 제한 시간 30초 동안 가장 높은 점수를 얻는 것이 목표입니다.

## 2. 조작 방식 (포즈 인식)
플레이어는 **고개 기울이기(Head Tilt)** 를 통해 바구니를 좌우로 이동시킵니다.

*   **CENTER (기본)**: 바구니 정지 또는 중앙 유지 (정면 응시)
*   **LEFT (고개 왼쪽으로 까딱)**: 바구니를 **왼쪽**으로 이동
*   **RIGHT (고개 오른쪽으로 까딱)**: 바구니를 **오른쪽**으로 이동

## 3. 아이템 정의

| 아이템 | 종류 | 효과 | 이미지(예시) |
| :--- | :--- | :--- | :--- |
| **사과 (Apple)** | 점수 | +10점 | 🍎 |
| **오렌지 (Orange)** | 점수 | +20점 | 🍊 |
| **바나나 (Banana)** | 점수 | +30점 | 🍌 |
| **폭탄 (Bomb)** | 감점 | **-50점** | 💣 |

## 4. 게임 규칙

### 기본 진행
*   아이템은 무작위 위치에서 생성되어 아래로 떨어집니다.
*   플레이어는 바구니를 아이템 위치로 이동시켜 받아야 합니다.
*   **제한 시간은 30초**입니다.

### 점수 규칙
*   과일을 받으면 점수가 추가됩니다.
*   **폭탄을 받으면 50점이 차감**됩니다. (점수는 음수가 될 수 있음)

### 게임 종료 조건
*   **30초가 지나면** 타임오버로 게임이 종료되고 최종 점수가 표시됩니다.

## 5. Teachable Machine 모델 클래스 (Labels)
